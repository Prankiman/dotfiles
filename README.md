# DOTFILES for user configuration

## REQUIREMENTS:

* git 
* powerlevel10k (used for zsh configuration)
    
### activating powerlevel10k:

run ``` mv powerlevel10k ~/powerlevel10k && echo 'source ~/powerlevel10k/powerlevel10k.zsh-theme' >>~/.zshrc ``` and open a new terminal

**or** run
``` 
    git clone git@github.com:romkatv/powerlevel10k.git ~/  
    echo 'source ~/powerlevel10k/powerlevel10k.zsh-theme' >>~/.zshrc 
```
and open a new terminal
